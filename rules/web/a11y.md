# Accessibility (a11y) Rules

Practical accessibility patterns for React and Next.js. WCAG 2.2 AA compliance.

## Semantic HTML First

Use the right element. Semantic HTML is free accessibility.

- `<button>` for actions, `<a href>` for navigation — never `<div onClick>`
- `<nav>`, `<main>`, `<header>`, `<footer>`, `<aside>` for landmark regions
- `<ul>`/`<ol>` for lists, not divs with bullets
- `<table>` only for tabular data — use `<caption>` and `<th scope>`

## Form Labeling

Every input needs a label:

```tsx
// Correct: visible label
<label htmlFor="email">Email</label>
<input id="email" type="email" />

// Correct: aria-label for icon-only inputs
<input aria-label="Search" type="search" />

// Wrong: placeholder is not a label
<input placeholder="Email" />
```

## ARIA Usage

- ARIA supplements semantics — never overrides them
- Do not add `role="button"` to `<div>` — use `<button>`
- `aria-label` on interactive elements without visible text
- `aria-describedby` for supplementary instructions
- `aria-expanded` + `aria-controls` on disclosure triggers (accordion, dropdown)
- `aria-live="polite"` for dynamic content updates (errors, toasts)
- `aria-invalid="true"` + `aria-describedby` for form error messages

## Keyboard Navigation

- All interactive elements reachable via Tab
- Logical tab order — do not use positive `tabIndex` values
- `tabIndex="-1"` for programmatic focus only
- `Escape` closes modals, dropdowns, tooltips
- Arrow keys navigate within composite widgets (menu, tab list, combobox)
- Visible focus indicator — never `outline: none` without an equivalent replacement

## Focus Management

- When opening a modal: move focus to first focusable element inside
- When closing a modal: return focus to the trigger element
- Trap focus inside open modals — prevent Tab from escaping
- After route changes: move focus to `<h1>` or page content region

## Images and Icons

- `<img>` always has `alt` — empty string `alt=""` for decorative images
- SVG icons used decoratively: `aria-hidden="true"`
- SVG icons conveying meaning: `role="img"` + `aria-label`

## Color and Visual

- Color must not be the sole means of conveying information (WCAG 1.4.1)
- Minimum contrast 4.5:1 for normal text, 3:1 for large text (WCAG 1.4.3)
- Text over images: use a contrast overlay, not raw overlay

## Testing

```bash
# Automated: catches ~30% of issues
npx axe-cli https://localhost:3000
# Or in tests:
import { axe } from "vitest-axe";
expect(await axe(container)).toHaveNoViolations();
```

Manual: screen reader (VoiceOver, NVDA), keyboard-only navigation, zoom to 200%.
